周蟒简史
========
2006 年底(12/22)，gasolin 在 python-cn group 上发表了一篇 "关于中文编程"(1) 的主题，引发了一波是否需要中文编程语言的讨论, 并引出了 HYRY 的中文字符串替代脚本。


HYRY 的原始版中文字符串代换脚本档案相当小，使用 utf-8 编码，依靠外挂 ini 字典档来转换中文字, 并且提供了部份未规范的简体中文关键词，可以算是周蟒的原型。


此后中文编程并没有立即的进展。 直到来年(2007) 8 月， gasolin 在吸收他前几个参与的开放原始码项目的经验(主要是 TurboGears )后，以 HYRY 的中文字符串代换脚本为概念中心，将原本简单的中文字符串代换脚本发展成完整的中文编程语言工具，并于 2007/8/9 发布了完整支持繁简中文编程的第一个版本 zhpy 0.2 版 (gasolin 将原本 HYRY 的脚本视为 0.1 版)。


0.2 版的特色是可以直接使用 python 的 easy_install 命令在线安装，简化了安装困难度; 并加入内建的繁简关键词对应字典, 提供中文编程统一的标准关键词参照。


接着三个月间， zhpy 以几乎每4天一个新版本的速度发布新版，每个释出版本都提供更多与 Python 语言兼容的中文编程功能。 在这过程中为了容易称呼，将 zhpy 的中文名称定名为「周蟒」。


周蟒在 0.5 版后陆续加入了一些工具。 让周蟒中文程序可以直接转换纯 Python 程序代码，以应用在一般的 python 程序中。


2007/10/2 ，周蟒发布了 1.0 版本，提供 API 文件、完整的覆盖测试、中文脚本、模块中文化插件。
