zhpy
====
zhpy 是“周蟒”的代名称。


zhpy是中文版的巨蟒，对台湾和中国大陆的初学者都有好处：用我们的母语学习 Python。


安装
====
安装 zhpy 的说明书: https://github.com/gasolin/zhpy/blob/wiki/DownloadInstall.md


打开命令行，输入当前源目录，然后键入以下内容命令（Linux）:

$ python setup.py install


基本玩法
========
点击链接： https://github.com/gasolin/zhpy/blob/wiki/BasicUsage.md


要玩zhpy，你甚至不需要安装它。

你所需要做的就是遵循3步指南:

1 .下载源代码包
2 .用zip解包工具展开源代码包
3 .运行:

$ python [路径]interpreter.py


然后你得到了可用的 zhpy 解释器！

验证安装
========
你可以用以下方法来确认是否已安装成功。
Linux 开启一个命令行 (Shell) 程序 (如 konsole 或 gnome-terminal)， 输入命令 'zhpy -V'。

$ zhpy -V
zhpy 1.x on python 2.x.x

现在我们假设你也已经在你的操作系统上准备好了 Python 与周蟒语言软件。
接着我们就来写我们的第一个 Python 程序吧。
